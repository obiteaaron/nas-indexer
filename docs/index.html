快速开始

三步启动，即可使用

1

克隆仓库

从 GitHub 克隆项目到本地

2

安装依赖

运行 npm install 安装所需依赖

3

启动服务

运行 npm start，访问 localhost:3000

命令行快速启动
# 克隆仓库
git clone https://github.com/obiteaaron/nas-indexer.git

# 安装并启动
cd nas-indexer
npm install
npm start

# 访问 http://localhost:3000
Docker 一键部署
# 构建镜像
docker build -t nas-indexer .

# 运行容器
docker run -d -p 3000:3000 -v nas-indexer-data:/app/profiles nas-indexer

# 或使用 docker-compose
docker compose up -d
