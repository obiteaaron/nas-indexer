# NAS 文件索引工具

一个完整的 NAS 文件管理系统，支持文件索引、查询、预览和基础文件操作。基于 Vue 3 + SQLite 构建现代化 Web 界面。

**版本: v1.3.0**

## 功能特点

### 核心功能
- 📁 **目录扫描** - 扫描指定路径，生成文件索引
- 🗄️ **SQLite 存储** - 高效的数据存储和查询
- 🔍 **高级搜索** - 多条件筛选、分类过滤、排序、搜索历史
- 📊 **统计分析** - 分类分布、文件数量、存储占比

### 文件管理
- 📍 **文件定位** - 一键打开文件所在目录（跨平台支持）
- ✏️ **重命名** - 在线重命名文件
- 📋 **复制/移动** - 文件复制和移动操作
- 🗑️ **删除文件** - 支持安全删除（移至回收站）
- ⭐ **收藏功能** - 标记常用文件，快速访问
- 🏷️ **自定义标签** - 多标签、分组、颜色、批量打标

### 标签系统
- 📂 **标签分组** - 按类型/年份/状态等分组管理标签
- 🎨 **标签颜色** - 自定义标签颜色，视觉区分更清晰
- ✅ **批量打标** - 选中多个文件一次性添加标签
- 🔍 **标签筛选** - 按标签组合筛选文件

### 文件分类
- 📂 **后缀分类** - 根据文件扩展名自动分类（视频、音频、图片、文档、字幕）
- 📁 **目录分类** - 根据目录路径前缀分类，优先级高于后缀规则
- ⚙️ **自定义规则** - 在设置页面自定义分类规则和目录映射

### 文件预览
- 🎬 **视频播放** - 流式播放，支持拖动进度
- 🖼️ **图片预览** - 直接显示图片内容
- 🎵 **音频播放** - 在线播放音频文件
- 📄 **PDF 查看** - 内嵌 PDF 预览器
- 📝 **文本预览** - 支持 .txt 纯文本文件预览
- 📖 **Markdown 渲染** - 支持 .md 文件实时渲染预览
- 📷 **缩略图** - 列表页显示图片缩略图，支持大小限制配置
- 🎞️ **视频元数据** - 显示视频时长和分辨率信息
- 🔍 **大图预览** - 鼠标悬停缩略图显示大图预览

### 用户体验
- 🌙 **暗色主题** - 支持浅色/深色主题切换，自动保存偏好
- 📈 **可视化图表** - 饼图展示分类分布和存储占比
- 🔎 **高级搜索** - 文件大小范围、修改时间范围筛选
- 🔌 **NAS 状态检测** - 扫描前检测 NAS 路径是否可达
- 📤 **导出功能** - 支持导出文件列表为 Excel
- 🔔 **浏览器通知** - 扫描完成时浏览器桌面通知
- ⚡ **性能优化** - 批量标签查询、虚拟滚动、API 缓存

### 其他
- ⏰ **定时扫描** - 支持 Cron 表达式配置自动扫描
- 🧠 **行为追踪与推荐** - 记录用户行为，生成个性化推荐

## 安装

```bash
npm install
```

## 使用

### 1. 启动服务

```bash
npm start
```

服务默认运行在 `http://localhost:3000`

### 2. 配置扫描路径

访问 Web 页面 → 设置 → 配置扫描路径

添加你的 NAS 挂载路径：
- Windows: `D:\NAS` 或 `\\NAS\Share`
- macOS/Linux: `/mnt/nas` 或 `/Volumes/NAS`

> **📝 关于 NAS 账密**：本应用暂不提供配置 NAS 账户密码的功能。
> 请在 Windows 系统中先保存 NAS 的凭据（访问网络路径时勾选"记住我的凭据"），
> 或在 Windows 凭据管理器中添加 NAS 凭据，之后直接输入路径即可访问。

### 3. 执行扫描

点击"立即扫描"按钮，扫描完成后即可在"文件列表"中查看所有文件。

## 注意事项

- NAS 关机时扫描会跳过，不会报错
- NAS 凭据需在 Windows 系统中预先配置，本应用不管理 NAS 登录认证
- 文件操作需要确保目标路径可访问
- 大文件扫描可能耗时，建议使用定时扫描
